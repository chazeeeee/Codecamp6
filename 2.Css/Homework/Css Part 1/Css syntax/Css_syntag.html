CSS syntax ประกอบด้วยอะไรบ้าง

รูปแบบของคำสั่ง CSS นั้นจะประกอบไปด้วย 3 ส่วน คือ
1.Selector
คือ ส่วน HTML tag ที่เราต้องการจะให้แสดงผล เช่น <h1> ก็จะแสดงผลในส่วนของชื่อเรื่อง
2.Property
คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร , ขนาดตัวอักษร
3.Value
คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีน้ำเงิน , ตัวอักษรขนาด 12 px

การเขียน CSS มีกี่แบบอะไรบ้าง

การเขียน CSS มีได้หลายรูปแบบในการจัดเก็บโค้ด หลัก ๆ ก็จะมีอยู่ 3 รูปแบบ ดังนี้

1.Inline Style Sheet
เป็นรูปแบบการเขียน CSS โดยการฝังลงไปใน line บรรทัดของ TAG HTML โดยการใช้ Attribute style แล้วตามด้วย value เช่น
<h1 style="font-size:20px;color:#00ffff;" >test</h1>
คือการเปลี่ยนขนาดตัวหนังสือเป็น20px และเปลี่ยนสีตัวหนังสือ tag h1 เป็นสีฟ้า
2.Internal Style Sheet
เป็นการเขียน CSS โดยการฝังรวมไปกับโค้ดของ HTML ซึ่งโค้ดของ CSS จะอยู่ในไฟล์เดียวกันกับเว็บเพจที่มีการเรียกใช้งาน CSS นั้น แต่มีการแยกออกมาอยู่ภายใน TAG style เช่น
<html>
   <head>
     <style>
        h1  {font-size:20px;color:#00ffff;}
     </style>
   </head>
   <body>
        <h1>test</h1>
   </body>
</html>
ซึ่งการแสดงผลจะเหมือนกับข้อที่1 โดยสามารถใช้ครั้งละมากๆ อ่านง่ายและเป็นสัดส่วนกว่าแบบแรก
3.External Style Sheet
สำหรับรูปแบบการเขียนในลักษณะ External Style Sheet จะเป็นรูปแบบที่เราสามารถเขียนโค้ดของ CSS แยกเก็บออกไปเป็นไฟล์เก็บไว้ภายนอกไฟล์เว็บเพจได้ โดยที่เราสามารถเรียกใช้ CSS ที่แยกเก็บไว้ได้ใน 2 รูปแบบ คือ 1. Linking Style Sheet 2. Import Style Sheet ไฟล์ CSS ที่เราแยกเก็บไว้นี้จะมีนามสกุลไฟล์เป็น *.css เช่น mystyle.css เป็นต้น
<html>
   <head>
     <link rel="stylesheet" href="mystyle.css">
   </head>
   <body>
        <h1>test</h1>
   </body>
</html>
ส่วนข้างล่างเป็นการสร้างไฟล์ที่ชื่อว่า mystyle.css
body {
background-color: lightblue;
}

h1 {
color:#00ffff;
font-size:20px;
}
