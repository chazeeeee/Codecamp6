property และ value คืออะไรแตกต่างกันอย่างไร

property

คือ ส่วนของคุณสมบัติที่เราต้องการให้แสดง เช่น สีตัวอักษร,ขนาดตัวอักษร,ความกว้าง,ความยาว

Value

คือ ค่าที่เราต้องการให้แสดง เช่น ตัวอักษรสีน้ำเงิน , ขนาดตัวอักษร โดยกำหนดส่วนใหญ่กำหนดเป็นตัวเลข

property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง

มี8กลุ่ม ได้แก่

CSS background Property
      background-color

CSS height Property
      height

CSS width Property
      width

CSS font Property
      font-style,font-size

CSS text-align Property
      text-align: left,right,center

CSS border Property
      border-width,border-style,border-color

CSS padding Property
      padding-top,right,bottom,left

CSS margin Property
      margin-top,right,bottom,left

value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร

Absolute length units
Unit Name Equivalent to
cm Centimeters 1cm = 96px/2.54
mm Millimeters 1mm = 1/10th of 1cm
Q Quarter-millimeters 1Q = 1/40th of 1cm
in Inches 1in = 2.54cm = 96px
pc Picas 1pc = 1/6th of 1in
pt Points 1pt = 1/72th of 1in
px Pixels 1px = 1/96th of 1in

Relative length units
Unit Relative to
em Font size of the parent, in the case of typographical properties like font-size, and font size of the element itself, in the case of other properties like width.
ex x-height of the element's font.
ch The advance measure (width) of the glyph "0" of the element's font.
rem Font size of the root element.
lh Line height of the element.
vw 1% of the viewport's width.
vh 1% of the viewport's height.
vmin 1% of the viewport's smaller dimension.
vmax 1% of the viewport's larger dimension.
